FAQ

  • COMMENT PUIS-je utiliser mes heticoins ?

    • - Les heticoins servent pour le moment à acheter des avatars sur le shop. Ils servent aussi en théorie à acheter de nouveaux modèles pour le pixel drawer mais cette fonctionnalité est en développement.


  • comment fonctionne le shop ?

    • - Le shop sert à acheter de nouveaux avatars. Il suffit de cliquer sur le visage de l'avatar choisit pour l'ajouter à son inventaire consultable dans l'encart "profil". Toute ressemblance avec des personnes existantes n'est absolument pas fortuite.


  • comment puis-je réinitialiser mon compte ?

    • - Rien de plus simple, le bouton "réinitialiser le compte" tout en bas de cette page. Si vous le souhaitez, vous pouvez aussi consulter ce site sur un autre navigateur ou bien une autre session de votre navigateur actuel.


  • comment puis-je reporter des bugs sur le site ?

    • - Si vous repérez un bug sur le site, veuillez contacter :

    • Si c'est un problème lié au JavaScript de la page, le MASTERMIND ou le CLICKER: contactez GABRIEL GOLDBRONN sur son linkedin accessible ici

    • Si c'est un problème lié au HTML / CSS de la page, le PIXEL-DRAWER ou le PUISSANCE 4 : contactez DORIAN CANONNE sur son linkedin accessible ici
